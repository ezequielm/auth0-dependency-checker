# Auth0 Dependency Checker

## Requirements
- Node.js v4.2.6
- Gulp v3.9.0

### Setup

Rename `config.json.sample` to `config.json` and update the attribute values
```bash
mv config.json.sample config.json
```

### Run api

Install dependencies
```sh
$ sudo npm install
```

Run
```sh
$ gulp
```

### Run tests

Run
```sh
$ make test
```
